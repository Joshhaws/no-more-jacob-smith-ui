# Strava Themed React App

A lightweight React app with a cool Strava-themed design featuring an activity table.

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Features

- Clean, modern Strava-inspired design with orange/red gradient theme
- Responsive activity table
- Lightweight and fast
- Built with Vite for optimal performance
